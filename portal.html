FlyOS

一个使用 Vue.js 和 Tailwind CSS 构建的现代开源 Windows 11 克隆。

在线演示

关于项目

FlyOS 是一个雄心勃勃的项目，旨在在 Web 浏览器中复制 Windows 11 美观直观的用户界面。它利用现代 Web 技术创造了快速、响应迅速且功能丰富的桌面体验。该项目使用 `Vite` 进行了初始化，并基于组件化架构构建，使其模块化且易于维护。

主要特性

Windows 11 风格用户界面

对 Windows 11 桌面、开始菜单、任务栏和小部件的忠实再现。

基于组件的架构

使用 Vue 3 组件构建，以实现模块化和可重用性。

集中式状态管理

使用 Pinia 进行状态管理，以实现可预测的状态转换。

窗口管理器

功能齐全的可拖动和可调整大小的窗口，带来逼真的桌面感觉。

内置应用程序

包括浏览器、记事本、终端等基本应用程序套件。

主题化

内置对浅色和深色模式的支持，提供个性化体验。

技术栈

Vue 3 Vite Pinia Tailwind CSS TypeScript
